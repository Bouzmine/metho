des livres reconnus par l'application

Selon un test approfondi mené sur les livres de la bibliothèque de l'École d'Éducation de McMasterville, 75 % que vous balayerez avec cette application seront reconnus et les données sur ces livres seront remplies.

Balayez le code sous la mention ISBN. Assurez-vous d'être dans un environnement assez éclairé.
